simple MC: a very simple MC transport code. Transports a particle in 1D, non-relativistic, on a two-cell mesh. Accumulates statistics as it goes.  

Files
-----
MC.hs: code
DataTypes.hs: data type definitions
Constants: physical and computational constants

Extras
------
Setup.hs, mc.cabal: attempt to cabalize 

To build: try 

cabal install 

End of file.
